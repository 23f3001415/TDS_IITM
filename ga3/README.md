# IITM TDS GA3 Solutions

This folder tracks GA3 assignment solutions question-by-question.

- `q1/README.md`: ELI15 solution and final submission answer for Question 1
- `q2/main.py`: FastAPI sentiment endpoint using structured outputs
- `q2/README.md`: ELI15 run/deploy steps and submission format
- `q2/endpoint_url.txt`: current public `/comment` URL
- `q2/start_all.ps1`: one-command start for API + tunnel
- `q2/stop_all.ps1`: one-command stop for q2 processes
- `q3/main.py`: FastAPI code interpreter with AI error-line analysis

- `q3/endpoint_url.txt`: current public `/code-interpreter` URL

- `q3/stop_all.ps1`: one-command stop for q3 processes
- `q4/README.md`: ELI15 solution and proof links for scheduled GitHub Action
- `q4/.github/workflows/daily-commit.yml`: daily cron workflow that commits automatically
- `q5/README.md`: ELI15 solution with workflow status badge
- `q5/.github/workflows/ci.yml`: CI workflow used for badge status
- `q6/README.md`: ELI15 solution with final Google Dork query
