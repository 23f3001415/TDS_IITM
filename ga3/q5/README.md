## ELI15 Step-by-Step

1. Create a GitHub repository.
2. Inside it, create `.github/workflows/ci.yml`.
3. Add `workflow_dispatch` so you can run it manually.
4. Push the workflow to GitHub.
5. Go to **Actions** and run the workflow once.
6. Confirm the workflow run is successful (green check).
7. Open the workflow page menu (`...`) and copy **Create status badge** markdown.
8. Paste the badge markdown at the top of `README.md`.
9. Push `README.md` changes.
10. Submit the repository URL.

## Workflow File

- `.github/workflows/ci.yml`
